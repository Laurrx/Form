SCALA DE INCAPACITATE BARTHEL

Masoara performantele pacientului in zece activitati ale vietii cotidiene in functie de ajutorul exterior necesar. Scorul maxim este 100 pct si corespunde unei autonomii complete. Scorul 60 semnifica "independenta asistata", iar cele 75 de pct cvasiindependenta

Activitate Descriere Scor
1. Alimentatia Independent: Se poate servi de tacamuri / Are nevoie de ajutor pentru a taia alimentele / Dependent
2. Baia Independent: Face baie fara ajutor / Dependent
3. Toaleta Personala Independent: Isi poate spala fata, mainile, dintii, isi poate pieptana parul, se poate barbierii, etc. / Dependent
4. Imbracatul Independent: Isi poate pune hainele, le poate scoate, isi poate innoda sireturile / Are nevoie de ajutor / Dependent
5. Control intestinal Fara probleme / Probleme ocazionale / Incontinenta
6. Controlul vezical Fara probleme / Probleme ocazionale (maximum 1 episod de incontinenta pe zi) sau are nevoie de sprijin in realizarea acestuia / Incontinenta
7. Transferul la toaleta Independent pentru a merge la toaleta / Are nevoie de ajutor in a merge la toaleta, dar se poate spala singur / Dependent
8. Transferul in pat sau fotoliu Independent in deplasarea de la scaun spre pat / Ajutor minim / Ajutor maxim pentru transfer, dar este capabil sa se aseze singur / Dependent
9. Mersul Independent - se poate deplasa singur 50 m / Merge cu ajutorul unui dispozitiv/cu sprijinul unei persoane / Utilizeaza singur fotoliul rulant / Dependent
10. Urcatul scarilor Independent in urcarea sau coborarea scarilor / Are nevoie de ajutorul unui dispozitiv sau al unei persoane / Dependent

INDEXUL IADL

Indexul IADL sau ADL Instrumental este de fapt o scala ce cuprinde 8 itemi, de fapt diverse activitati curente, instrumentale specific traiului zilnic. Pentru fiecare actiune pacientul este punctat in functie de performanta in realizarea cerintei.

Afectiuni somatice sau neurologice severe care pot potenta deficitul locomotor SCOR ADL

Activitate Descriere Scor
1. Igiena corporala Autonomie / Ajutat partial / Dependent
2. Imbracare Autonomie / Autonomie pentru imbracare, dar ajutor pentru incaltare / Dependent
3. Mersul la toaleta Autonomie / Ajutat / Ajutat la pat
4. Continenta sfincteriana Continent / Incontinenta ocazional / Incontinent
5. Alimentatia Mananca singur / Ajutat pentru taiat cu cutitul / Dependent

Clasificare: st. I (10) - autonomie; st. II (8-10) - cvasiindependent; st. III (3-8) - independenta asistata; st. IV (0-3) - dependenta totala, necesita asistent personal.
